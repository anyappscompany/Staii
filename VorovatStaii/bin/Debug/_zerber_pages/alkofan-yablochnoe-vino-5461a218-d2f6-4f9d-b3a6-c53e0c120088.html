Динамика рынка рекламы в поисковых системах требует ежемесячного анализа. Запросы, которые вы проверили и отобрали, уже сегодня могут быть неэффективны. Ситуация меняется ежедневно и аналитическая информация - Ваш главный козырь. Наш сервис позволяет наблюдать за конкурентами, анализировать изменения на рынке, своевременно реагировать на такие изменения и, как следствие, эффективно управлять рекламными кампаниями.

Аналитика доменов:

1Контекстная реклама (direct&adwords):

  • Ключевые запросы, по которым идут показы объявлений у домена и их показатели: частотность запроса, % трафика от общего по контексту, СРС (1, 2, 3)
  • Объявления: заголовки, тексты и по каким ключевым запросам идут показы, URL
  • Оценочный бюджет и трафик
  • Конкуренты по пересечению облака запросов и сводная аналитика по ним

2Органический поиск (Yandex&Google):

  • Ключевые запросы домена (топ20) и их показатели: частотность запросов, % трафика от общего по органике, СРС (1, 2, 3), заголовки, сниппеты
  • Распределение запросов по страницам
  • Оценка трафика
  • Конкуренты по пересечению облака запросов и сводная аналитика по ним

3Видимость домена (топ20):

  • Органический поиск
  • Контекстная реклама

Анализ «Ключевых запросов»:

1Сводные данные:

  • Показов в месяц
  • Рекламодателей
  • CPC, спец. размещение
  • CPC, 1-е место (гарант.)
  • CPC, гарантир-е показы
  • Результатов в выдаче
  • Что еще искали с запросом: перечень ключевых запросов и аналитика по ним: запросов в месяц, СРС, результатов в поиске

2Контекстная реклама:

  • Объявления по запросу: заголовок, текст, домен, его показатели, ссылка на лендинг
  • Облако ключевых запросов, по которым идут показы доменов из топ20 контекста по ключевому алкофан яблочное вино запросу.

3Органический поиск:

  • Результаты органического поиска по ключевому запросу, заголовки, сниппеты и аналитические данные по доменам: трафик и кол-во запросов в поиске
  • Облако ключевых запросов лендингов из топ20 и их аналитические данные: СРС (1, 2, 3), запросов в месяц, результатов в поиске
